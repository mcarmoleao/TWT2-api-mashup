## Elementos do Grupo

## Tecnologias e APIs utilizadas
- Node.js, Express
- MongoDB Atlas
- Unsplash API

## Instalação e configuração
1. Clonar o repositório  
2. Instalar dependências: `npm install`  
3. Criar um ficheiro `.env` com as variáveis:  
   - MONGO_URI=...  
   - SESSION_SECRET=...  
   - UNSPLASH_KEY=...

## Como executar localmente
- `node server.js`
